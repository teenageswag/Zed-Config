# Minimal Blur Theme

Минималистичная темная тема для редактора Zed с акцентами в стиле Nuclear (Золотой/Зеленый) и поддержкой эффекта размытия.

Цветовая палитра адаптирована для снижения нагрузки на глаза: мягкий темно-серый фон и приглушенный текст.

## Особенности

- **Фон:** Мягкий темный (`#141414`) с прозрачностью.
- **Текст:** Светло-серый (`#BFBFBF`) для комфортного чтения.
- **Акценты:** Золотой (`#CCB333`) для ключевых слов и Зеленый (`#73B82E`) для строк.
- **Blur:** Поддержка эффекта размытия окна.

## Установка (Dev версия)

1. Клонируйте или скачайте этот репозиторий.
2. Откройте **Zed**.
3. Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac) и введите **"extensions"**.
4. Нажмите кнопку **"Install Dev Extension"** вверху справа.
5. Выберите папку с этой темой.

## Настройка

Для включения эффекта размытия убедитесь, что в ваших настройках (`settings.json`) включена поддержка прозрачности (если ваша ОС это поддерживает):

```json
{
  "theme": "Minimal Blur",
  "window_background_appearance": "blurred"
}
```

## Настройка Блюра (Раздельная)

Вы можете настроить прозрачность отдельно для боковых панелей (файлы) и редактора (код).

### 1. Боковая панель и общие окна (95%)

- Параметр: `"background"`
- Код: `#141414f2`
- `f2` = 95% непрозрачности

### 2. Редактор кода (95%)

- Параметры: `"editor.background"` и `"editor.gutter.background"`
- Код: `#141414f2`
- `f2` = 95% непрозрачности

### Таблица значений

- `ff` = 100% (Нет блюра)
- `f2` = 95% (Почти нет блюра)
- `e6` = 90% (Легкий блюр)
- `cc` = 80% (Средний блюр)

*Меняйте последние два символа в соответствующих строках файла `themes/minimal_blur.json`.*
